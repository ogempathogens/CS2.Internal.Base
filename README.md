# Internal C++20 base for counter-strike 2

Features:
- Hooks using MinHook
- Draws an ImGui window inside the game
- Basic sig scanning

## How to build:
- Everything should work out of the box, if you aren't using VS/don't want to use the project files, make sure you are compiling with C++20

## How to run:
- After building the project injection can be done either via manual mapping or lla (-insecure/-allow_third_party_software/dll at windir).
